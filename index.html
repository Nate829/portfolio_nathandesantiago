    This web portfoilo includes examples of my work in Mr. Lanus' Web Design class. It also includes my reflections (below) on what I learned in each unit of the course.

    Unit 1: HTML

  • What do you think are the most important vocabulary terms (or key concepts) in this unit? What do you think that they mean?

      • I think that the most important vocabulary terms are : tags, DOCTYPE, ul, ol, h1, head, and etc.They are used to create links, different fonts, add text, and many other different tags.

  • What do you think are the most important skills that you learned in this unit of study?

      • I learned how colors play an important part in the way a website can turn out, and how to use tags like < p >, < h1 >, < main >, and many more.

  • In what ways have you gotten better at this kind of work?

      • I've learned about how to use tags, and how they work.I also learn how to use html and how to start a website from scratch.

  • In what ways do you think you need to improve?

      • I need to have more patience and creative thinking when it comes to web design, because this class requires a lot of patience and thinking.

  • What problems did you encounter while you were working on this piece? How did you solve them?

      • I don't really run into that much problems. It's more of a bad habit, I would forget to add the end dash on almost every end tag.

  • What was especially satisfying to you about either the process or the finished product?

      • I really like the finished product because it looks like an actual website and that's the look I was going for.

  • If you were the teacher, what comments would you make about this piece?

      • Nothing, I would admire this work and I just would say “This piece is beautiful.”
